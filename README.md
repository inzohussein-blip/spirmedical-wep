# Spir Medical سبير الطبية

Iraqi Home Healthcare Web App — تطبيق ويب عراقي للخدمات الصحية المنزلية

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/inzohussein-blip/Spir_Medical)

## Features
- 🏠 Home Screen with Stories, Banners, Services Grid
- 🤖 AI Symptom Checker
- 🏥 14 Real Iraqi Hospitals Directory
- 💳 Payment (Zain Cash, Mastercard, Wallet, Cash)
- 🛡️ Micro Insurance Plans
- 🏅 Gamification Badges (15 badges)
- ⭐ Loyalty Program (4 tiers)
- 🔔 Notifications Center
- ⚙️ Settings (Language, Notifications, Theme)
- 📱 PWA - Install on phone as native app

## Tech Stack
- Pure HTML/CSS/JS (No framework needed)
- PWA with Service Worker
- Vercel for hosting
- Supabase for backend (optional)
